# 📸 Pasta para Fotos do Casal

## ✨ Como Usar

1. **Coloque suas fotos nesta pasta**: `images/casal/`
   - As fotos devem ser em formato `.jpeg` ou `.jpg`

2. **Nomeie as fotos** de forma sequencial:
   - `1.jpeg`
   - `2.jpeg`
   - `3.jpeg`
   - ... até `21.jpeg` (conforme o número de fotos no mosaico)

3. **Foto de Sucesso** (IMPORTANTE):
   - A foto `25.jpeg` é especial
   - Ela é exibida quando alguém clica em "Pagamento Realizado"
   - **Escolha uma foto bem bonita e feliz do casal para esta!** 🎉

## 📋 Estrutura Esperada

```
images/
├── casal/
│   ├── 1.jpeg      (foto do mosaico)
│   ├── 2.jpeg      (foto do mosaico)
│   ├── 3.jpeg      (foto do mosaico)
│   ├── ...
│   ├── 21.jpeg     (foto do mosaico)
│   ├── 25.jpeg     (FOTO DE SUCESSO - exibida no modal)
│   └── qr-code.png (QR code de pagamento)
```

## 🎯 Dicas

- **Dimensão recomendada**: Pelo menos 300x300px (serão redimensionadas automaticamente)
- **Foto 25**: Use uma foto especial do casal - será exibida quando alguém confirma o pagamento!
- **Qualidade**: Fotos em boa resolução ficam melhores
- **Formato**: Use `.jpeg` ou `.jpg`

## 💡 O Que Acontece

- **Mosaico (fundo)**: Mostra as fotos 1 a 21 em um padrão de grade
- **Tela de Sucesso**: Quando alguém clica "Pagamento Realizado", a foto 25 é exibida em grande no modal
- **Celebração**: Aparece com animação elegante e confetes! 🎊
